Download Kernel from kernel.org
https://cdn.kernel.org/pub/linux/kernel/v4.x/linux-4.16.1.tar.xz

Untar Files for Compiling it
tar -xvf linux-4.16.1.tar.xz

Change Directory to linux -4.16.1

Make Configuration File using make menuconfig command.

Make Kernel and Modules using commands make and make modules respectively.

Install Kernel Modules and then Kernel by executing the commands sudo make modules_install
and sudo make install.
